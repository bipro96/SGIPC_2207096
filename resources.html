Resources

Everything you need to start and grow your competitive programming journey

Beginner Start Here

New to competitive programming? These resources will build your foundation in problem solving, basic algorithms, and coding fundamentals.

CF

Codeforces Edu — ITMO

Step-by-step course on basic algorithms and data structures with practice problems.
CS

CSES Problem Set

Comprehensive collection of competitive programming problems organized by topic.
US

USACO Guide

Structured learning path from Bronze to Platinum with modules and curated problems.
HR

HackerRank — Algorithms

Beginner-friendly environment with tutorials and easy-to-medium difficulty problems.

Intermediate Level Up

Ready to push your skills further? These resources cover advanced data structures, common patterns, and intermediate contest strategies.

CF

Codeforces

The largest competitive programming platform. Regular contests, Div 2/3 participation, and vast problem archive.
AC

AtCoder

Japanese platform known for high-quality problems. ABC and ARC contests are excellent for skill building.
CP

CP-Algorithms

Comprehensive reference for competitive programming algorithms with clear explanations and code.
LC

LeetCode

Excellent for interview prep and practicing standard patterns. Weekly contests and company-tagged problems.

Advanced Expert

For serious competitors aiming for ICPC Regionals, World Finals, and top ratings. Advanced topics, hard problems, and elite contest preparation.

90

Typical DP Contest

90 essential competitive programming problems covering a wide range of classic techniques and patterns.
OJ

OJ.uz

Archive of problems from prestigious contests like IOI, CEOI, and Balkan OI. High difficulty, high reward.
LB

Library Checker

Test your implementations of advanced algorithms against strict test cases. Great for verifying your code library.
EG

E869120's Blog

Curated problem lists and study plans from one of Japan's top competitive programmers.
